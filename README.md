# Blog CMS Angular

Angular 19 blog CMS frontend.

## Features

- Post management
- Comentario management
- Info page
- Modern Angular 19 with standalone components

## Tech Stack

- Angular 19 (standalone components)
- TypeScript
- SCSS

## Development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Migrated

- Upgraded from Angular 9 (JHipster 6.7) to Angular 19
- Migrated from NgModules to standalone components
- Modernized HTTP services

## Note

This is the frontend only. The backend (Java/Spring Boot) needs to be migrated separately using JHipster 8+.
